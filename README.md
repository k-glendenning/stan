Stanley Kudzu
=============

Stan is an IRC bot written primarily in awk. His chat module produces markov
chains when he is highlighted, and he learns from listening to the channels he
is in.

User commands
--------------

* `%cocain`
* `%nsa`
* `%police`
* `%status`

Administrative commands
-----------------------
* `%identify`
* `%join`
* `%nick`
* `%part`
* `%quit`
* `%reload`
* `%restart`
* `%say`
* `%sync`
